# G4web

Client for G4 project - a clone of the cgv.vn website

## Installation

```bash
git clone https://github.com/K63PTDAPMgroup4/G4web.git
cd G4web
```

## Usage

```bash
cp .env.example .env
```

Then open `.env` file and fill in the environment variables

for development:

```bash
npm install
npm run dev
```

for preview production:

```bash
npm run build
npm run preview
```
